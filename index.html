Who?

John Shea is a full-stack web developer, entrepreneur, and investor with a background in law and finance. He enjoys building rich, full-stack web applications that are a pleasure to use and solve concrete problems.

FlareGun

FlareGun (formerly <sos>) is a location-based help app for programmers.
Post an SOS request, and a nearby programmer will swoop in on feathery, axe-scented wings to help you in your distress.
SOS requests can be tagged with "skills" such as jQuery, Ruby, or MySQL, so as to match up a crisis with the right neck-bearded savior.

Fünke Wiki

Built as a project during General Assembly's Web Development Immersive, Fünke Wiki is a mobile-first collaborative encyclopedia built using Express, EJS, and Node.js.
The site follows Material Design guidelines, and features include version control, upvoting of drafts, secure user login, etc.
